Problem/Data Domain

For our general topic we will be looking at how we can improve the interactive system for music-based web applications. There are various client-side feature-related problems that can impact user experience. Some challenges that these music-based web applications like Spotify, Apple Music, and Soundcloud face are customization limitations, personalization and recommendations, and social features. This is due to making these music applications more engaging, user-friendly, and customizable to individual preferences, providing a comprehensive music experience for users. Addressing these client-side features requires ongoing development and testing efforts and prioritizing user feedback in order to release updates that resolve these issues and improve usability.

However, not all of these problems can be addressed as there are still some limitations and loose ends that these applications still have. Mainly pertaining to user usability and customization, other web-based music applications like Spotify and Apple music can have irrelevant music recommendations which can cause the user to receive poor suggestions which can lead to user dissatisfaction. For our application our goal is for our users to have full control over their customizations and preferences so that they are able to truly utilize certain features that are accustomed to their own personalization.
